## 快速上手

本节将介绍如何在项目中使用 d5c-map。

### 使用 vue-cli@3

我们demo展示项目中使用vue-cli构建

### 属性说明

| 属性名	     | 类型          | 默认值	      | 说明                           | 备注|
|:--------: |:------------: |:--------: |:------------------------------:  |:-----: |
| mid    | string         |  map | 地图id 唯一  | 全局唯一 |
| markData     | Array          | -  | 地图上标记点 | - |
| track     | Array           | -  | 行车轨迹  | - |
| showPaths     | Array          | -  | 轨迹回放  | - |
| drivingData     | Array          | -  | 线路规划  | - |
